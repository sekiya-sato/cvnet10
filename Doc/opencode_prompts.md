コピー＆ペーストして、そのまま使用できるように設計されています。

## 1. 新規画面作成

### 1.1 基本的なマスタメンテ画面


## Tips: 効果的なプロンプトの書き方

### ✅ DO（推奨）

1. **具体的な参照を含める**
   ```
   MasterTokuiMenteView.xaml と同じ構成で作成してください
   ```

2. **明確な制約を指定**
   ```
   Cvnet10Base は編集せず、Cvnet10DomainLogic に実装してください
   ```

3. **期待する成果物を明示**
   ```
   Mermaid形式のクラス図を含めてください
   ```

4. **検証方法を指示**
   ```
   作業完了後、Buildして結果を確認してください
   ```

### ❌ DON'T（避けるべき）

1. **曖昧な指示**
   ```
   画面を作って
   ```

2. **範囲が不明確**
   ```
   全部リファクタリングして
   ```

3. **禁止事項を無視**
   ```
   CodeShare に追加して（許可なし）
   ```

4. **検証方法が不明**
   ```
   よろしく（何を確認？）
   ```

## まとめ

このプロンプトライブラリを活用することで、OpenCodeでの作業効率が大幅に向上します。

**使い方のコツ：**
1. プロンプトをコピーして、`[...]` 部分を具体的な内容に置き換える
2. 必要に応じて要件を追加・削除する
3. 作業完了後は必ず Build 確認を含める
4. Read-Onlyプロジェクトの編集時は明示的に許可を記載
