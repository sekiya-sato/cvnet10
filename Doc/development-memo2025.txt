## MVVM

VM間のメッセージ通信

送り側：
	WeakReferenceMessenger.Default.Send(new SampleMessage("this is test !!!!!!"));

受け側：
	WeakReferenceMessenger.Default.Register<SampleMessage>(this, (r, m) =>
	{
		// メッセージを受信したときの処理
		HandleMessage(m);
	});
	private void HandleMessage(SampleMessage message) {
		// メッセージの内容を処理
		Console.WriteLine(message.Text);
		if (CurrentEdit == null)
			Current = new MasterShop();
		CurrentEdit.Code = message.Text; 
	}
	public class SampleMessage {
		public string Text { get; }

		public SampleMessage(string text) {
			Text = text;
		}
	}

- CreateSQL＜T＞ --- テーブル及びキー作成用SQL

- Sysman... --- システム管理系テーブル
- Master... --- マスターテーブル
- SubMst... --- 補助マスターテーブル
- ...Sub.. --- 関連サブテーブル
- ...Lnk.. --- 中間テーブル
- Trans... --- トランザクションテーブル
- Rireki... --- 履歴テーブル
- Shukei... --- 集計テーブル

	※名前に使う数字は0から付与する

- Id_テーブル名か俗称 --- 対象テーブルの連番Id
